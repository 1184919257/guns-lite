基本环境准备

本文档基于Intellij IDEA,Mysql,Maven，JDK8这四个基本工具， 当然你也可以用Eclipse开发工具。

Jdk

  • 请选用当前最新的版本,根据平台选用X64或X86版本的JDK8,并妥善安装

MySQL

  • Mysql官网
  • 下载页面, 选用5.7系列的最新版
  • 妥善安装,并设置密码为root

Intellij IDEA

  • Intellij IDEA官网
  • 下载页面
  • 请自行下载安装合适版本的IDEA（或者eclipse）
